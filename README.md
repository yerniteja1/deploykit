## Stack
- Frontend: React + TypeScript + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: Supabase (PostgreSQL)
- Auth: GitHub OAuth2 + JWT
- Real-time: Server-Sent Events (SSE)
- Deployment: Vercel + Render
